## 🚀 **Quick Start Guide**

### **Prerequisites**
```bash
Node.js >= 16.0.0
npm >= 8.0.0
Git >= 2.30.0
```

### **Installation & Setup**

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ashwanikumar0555/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm start
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
